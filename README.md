# Arduino Tian Weather Hub

![tianmain](https://cloud.githubusercontent.com/assets/10930201/26062891/e5da0e0e-3951-11e7-945a-605a0beeeaf1.png)

New connected devices are coming out everyday. Arduino joined the fray with the Yun, but a new challenger has appeared - the Arduino Tian!

Using a combination of python, classic Arduino sketches, the Lucky Shield, and Weather Underground, this tutorial will walk you through turning your Tian into a weather hub. But since Lucky shield has so many sensors, it could be much more!
